## Organisation des branches

Le dépôt est structuré autour de plusieurs branches principales :

- **main** : version finale du projet (vue d’ensemble)
- **rpi** : scripts et configurations liés au cluster de Raspberry Pi et aux calculs parallèles
- **website** : code de la plateforme web
- **doc** : documentation et rapports

## Objectifs du projet

- Mettre en place un cluster de Raspberry Pi fonctionnel  
- Implémenter plusieurs algorithmes de calcul parallèle
- Développer une interface web pour piloter les calculs
